## نصب

برای نصب دستور زیر را اجرا کنید

```bash
composer require bazofather/splus-php
```

## نحوه استفاده

```php
$instance = new Bazofather\SPlusPHP();
echo $instance->echo('Hello, Bazofather!');
```
